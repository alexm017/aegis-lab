# Aegis Lab Cybersecurity Team Website

Official website for the Aegis Lab high school cybersecurity team.

Live URL: **https://aegislab.ro/**

## Website Preview

![Aegis Lab website preview](assets/preview/website-preview.png)

## Main Pages

- `/` - Home
- `/members` - Team members
- `/events` - Upcoming and past events
- `/events/aegis-lab-kickoff` - Kickoff event page
- `/roadmap` - Competition roadmap
- `/gallery` - Team gallery
- `/team-ctf` - Team CTF link page
- `/dashboard` - Private admin dashboard

## Stack

- PHP
- Apache (`.htaccess` routes + rewrites)
- HTML/CSS/JS
